- Necessarios colocar GitBash e Github no computador

comandos para rodar o GIT com o hub.

echo "# Test_repositorio" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/VVelos0/Test_repositorio.git
git push -u origin main

Criar um repositorio já existente
git remote add origin https://github.com/VVelos0/Test_repositorio.git
git branch -M main
git push -u origin main

Para rodar os comandos GIT bash são 

git checkout -b "NovaBranch" = para rodar novas branches e sair da principal
git add . = Para colocar todos os arquivos
git commit -m "NovasMensagens" = para colocar mensagem no seu repositorio
git push origin "NovaBranch"= para fazer o commit nesta pasta dps commits
